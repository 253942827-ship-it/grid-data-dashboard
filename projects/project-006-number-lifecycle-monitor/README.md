# project-006-number-lifecycle-monitor — 号码生命周期监控看板

- **项目编号：** 006
- **项目名称：** 号码生命周期监控看板
- **创建日期：** 2026-07-09
- **项目目标：** 对号码全生命周期进行监控，包括出账监控、状态监控、佣金回款监控三大板块

## 目录结构

```
project-006-number-lifecycle-monitor/
├── README.md      # 本文件
├── data/          # 原始数据（号码清单、状态数据、佣金数据等）
├── src/           # 源代码
├── docs/          # 文档、监控看板
├── outputs/       # 交付物
└── work/          # 临时工作文件
```

## 监控看板三大板块

### 1️⃣ 出账监控
- 号码出账状态跟踪
- 费用支出与收入对比
- 出账异常预警

### 2️⃣ 状态监控
- 号码当前状态（正常/停机/销户等）
- 状态变更历史
- 在网时长统计

### 3️⃣ 佣金回款监控
- 佣金应收/实收对比
- 回款进度跟踪
- 逾期预警

## 数据来源

| 数据类型 | 说明 | 更新频率 |
|----------|------|----------|
| 号码基础信息 | 号码、入网时间、入网渠道、费用支出 | 首次导入 |
| 状态数据 | 号码当前状态、状态变更 | 定期更新 |
| 佣金数据 | 佣金金额、回款情况 | 定期更新 |
